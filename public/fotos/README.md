# Fotos dos Instrutores

Salve aqui as fotos dos 3 instrutores do curso com EXATAMENTE esses nomes:

- `lucas.jpg` — Lucas Pereira
- `murilo.jpg` — Murilo
- `joao.jpg` — João

**Especificações recomendadas:**
- Formato: JPG (mais leve) ou PNG
- Resolução mínima: 800×800px (quadrada, será cropada como quadrado no site)
- Peso máximo: 500KB cada (otimize em [tinypng.com](https://tinypng.com) se necessário)
- Boa iluminação, fundo neutro de preferência

Se quiser usar outros nomes ou formatos, ajuste os componentes em [`src/pages/index.astro`](../../src/pages/index.astro) e [`src/pages/sobre.astro`](../../src/pages/sobre.astro) (array `instructors`).
